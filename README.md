# Human Altruism

**El Altruismo Humano es la práctica de dar que comienza por entender quién eres tú — no por calcular dónde rinde más tu dinero.**

Framework independiente para personas que quieren dar bien. No afiliado a ninguna organización, fundación ni movimiento político.

🌐 [human-altruism.org](https://human-altruism.org)
